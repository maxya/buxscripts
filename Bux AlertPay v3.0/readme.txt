Installation:

1. Open up PHPMyAdmin or your favourite Database editing software.
2. Create an empty table and call it anything you like, we'll call ours "buxscript".
3. Open up the empty table and Import the database.txt file.

4. Locate and edit the config.php and /admin/config.php files.

5. Upload all the files to your server.

6. Load your index page and login with the default admin details (do not go to the admin section).

7. When logged in click the Admin link displayed on the main page this will open a new window.
